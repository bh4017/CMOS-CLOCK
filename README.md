# CMOS-CLOCK


# DESCRIPTION

A digital clock made with CMOS 4000-series logic ICs.  Design is based on work by San Bergmans at his website, here: http://sbprojects.com/projects/cmosclock/

PCB Design by Brian Hoskins: http://brianhoskins.uk
